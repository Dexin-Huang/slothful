Aidolon — Privacy Policy

Last updated 2026-04-07. Operated by Slothful.

Draft v0. This document is a starting template and has not yet been reviewed by counsel. It will be updated before public launch.

Who we are

Aidolon is a voice-first AI companion that runs on Ray-Ban Meta glasses paired with an iPhone app and a Mac-side reasoning agent. It is built and operated by an individual developer doing business as Slothful (the "operator", "we", "us"). Contact: dexin@slothful.ai.

What Aidolon does with data

Aidolon is designed to be a personal tool. The operator does not run a multi-tenant cloud service. Specifically:

  • Audio. The glasses microphone is used for wake-word detection and live conversation. Audio is streamed to Google's Gemini Live API for speech-to-text and reasoning. Aidolon does not retain audio recordings.
  • Camera frames. When the user invokes a vision request, individual frames from the glasses camera are sent to Gemini Live. Frames are not retained by Aidolon after the request completes.
  • Delegated tasks. When Aidolon delegates a task ("send a message", "what's on my calendar"), the natural-language description is forwarded over the user's local network to a Mac-side agent owned by the user. The agent runs on the user's own machine; the operator never receives this data.
  • Messages sent through Aidolon. Messages drafted by Aidolon and sent through Twilio RCS leave the user's device through Twilio's network. Twilio's privacy practices apply to those messages in transit. See Twilio's privacy notice.
  • Diagnostic logs. The Mac-side agent writes local log files for debugging. These never leave the user's machine.

Third parties

Aidolon transmits data to the following third parties on the user's behalf:

  • Google (Gemini Live API). Audio and camera frames for live AI processing. Terms.
  • Anthropic (Claude API, via the Claude Agent SDK). Delegated reasoning tasks. Privacy notice.
  • Twilio (RCS). Messages sent on the user's behalf. Privacy notice.
  • Meta (Wearables Device Access Toolkit). The Ray-Ban Meta glasses connection layer. Privacy notice.

Data retention

The operator does not run servers that retain user data. All user-facing state lives on the user's iPhone, Mac, or in the third-party services listed above, each of which has its own retention policy.

Children

Aidolon is not intended for use by children under 13.

Your choices

  • Disconnect the glasses or quit the iPhone app to stop all data flow at any time.
  • Stop the Mac-side agent process to revoke the iPhone's ability to delegate tasks.
  • Contact the operator to ask questions about this policy.
