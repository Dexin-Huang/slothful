# slothful

Public asset bucket for [Slothful](https://github.com/Dexin-Huang?tab=repositories&q=slothful) projects. Anything that needs to live behind a public URL goes here — brand assets, RCS sender images, hosted JSON, OG previews, whatever.

Serve raw files via:

```
https://raw.githubusercontent.com/Dexin-Huang/slothful/main/<path>
```

## Layout

One subfolder per consumer. Add a new one when you need it.
